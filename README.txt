This automation framework script was acted as template for Adaptavist automation engineer candidates to start writing automation for interview home task.

Please download chromedriver in here https://chromedriver.chromium.org/downloads if it not compatible with your chrome browser.

The components that has been used in building up this automation framework was :
-Java langugage
-Selenium Webdriver
-JUnit
-PageFactory
-Intellij
-Chrome
-Maven

This project contains sample solutions against the Herokuapp website scenarios as below:
- Login with valid credentials
- Login with invalid credentials

Step to run:
Please use 'mvn clean install' command from the terminal in order to download dependencies, build, and run the test
